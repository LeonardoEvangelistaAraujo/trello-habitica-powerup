How it works

Fully automatic
1
Connect Habitica once

Click Setup Habitica (board header or first card open), enter your User ID and API Token from Habitica → Settings → API. That's the only manual step.

2
Set a label to choose task type

Add one of these labels to a card before acting on it:

TODO One-off task — completes once and disappears from Habitica
DAILY Repeating task — resets each day, earn XP every time
HABIT Score + or − as many times as needed
No label? Defaults to To-Do.
3
Just click the action button — everything else is automatic

Open any card. The buttons on the right handle everything:

✓ Mark Done Creates the task in Habitica (if not yet) and immediately completes it. XP & Gold earned.
+ Habit Creates the habit task (if not yet) and scores it positively.
− Habit Creates the habit task (if not yet) and scores it negatively.

A green ⚔ Synced badge appears on the card face once the task is created.

4
Move a card to a Done list — auto-completes

If a card is in a list whose name contains Done, Complete, Finished, or Archive, opening the card automatically completes the linked To-Do in Habitica. No button click needed.

Dailies and Habits are not auto-completed on list move — use the card buttons for those.
5
Watch your stats update on the board

HP, XP, Gold, and Level are shown directly in the board toolbar. They refresh each time the board loads. Click any stat to open the full dashboard.

Storage: Card–task links are saved in Trello's own card data (t.set), shared across all browsers and board members. Credentials (User ID & API Token) are saved only in your browser's localStorage.

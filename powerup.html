How Card Sync Works

Trello → Habitica
1
Open any card

Click a Trello card to open it. You'll see a Sync to Habitica button in the card's button bar on the right.

2
Choose a task type

Select To-Do, Daily, or Habit — then click the matching button.

Tip: Add a label named TODO, DAILY, or HABIT to the card before clicking Sync and the type will be pre-selected automatically.
3
Task is created in Habitica

The card name becomes the task title. The card description becomes the task notes. A green ⚔ Synced badge appears on the card face.

4
Act on the synced task

Open the card again. The button now says Mark Done or Habit +/− depending on type:

To-Do Mark Done → completes the task in Habitica, earns XP & Gold
Daily Mark Done → checks off today's daily, earns XP & Gold
Habit + scores positively (XP up)  •  − scores negatively (HP down)
5
Unlink a card

Open the card → click the action button → scroll down to Unlink from Habitica. This removes the local mapping only — the Habitica task is kept.

Storage: The link between each Trello card and its Habitica task is stored in your browser's localStorage. It is specific to the browser and device you used to sync. Clearing browser data will remove all mappings.

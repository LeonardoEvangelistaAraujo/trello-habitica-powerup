🚀 How to use

  1. Enter your Habitica credentials above and click Save & Connect.
  2. Go to Trello Power-Ups and create a new Power-Up.
  3. Set the iframe connector URL to this page's URL (e.g. https://your-user.github.io/repo/powerup.html).
  4. Add the Power-Up to your Trello board.
  5. On any card, click Sync to Habitica to link it.
  6. Use labels TODO / DAILY / HABIT to set task type automatically.

🏷️ Label Guide

TODO One-off task, disappears when completed
DAILY Repeating task, resets every day
HABIT Score + or − any number of times

All data stored locally in your browser. No backend. No server.
